jQuery Open Carousel - Fullwidth Example

github: github.com/justinmc/jQuery-Open-Carousel



This example demonstrates the data-ocarousel-fullwidth configuration attribute, which dynamically sets each frame's width to the full width of the carousel window. This allows you to match your carousel content to the width of the carousel in dynamic situations, like a responsive site or a fullscreen carousel. Try changing the size of your browser window or rotating your mobile device and see how it adapts.
