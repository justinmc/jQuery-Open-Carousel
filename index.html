jQuery Open Carousel

by: Justin McCandless
github: github.com/justinmc/Scrolling-Image-Bar



Open but Easy

This project is an attempt to create a carousel that is easily used without touching any javascript, but also plenty extendible and easily hacked for anyone with a more crafty application.

Fit Whatever Content You Want

Wrap stuff in divs or use img tags directly, make every item the same size or each one unique, all will be handled intuitively.

And...


Features like easily adding indicator circles, scrolling by multiple slides at once, support for multiple instances, the ability to link to any slide...
